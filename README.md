# BTV-System-Design

> Official Enterprise Design System Website & Guidelines for Boonthavorn Retail Corporation.

[![Deploy BTV System Design to GitHub Pages](https://github.com/biggerbababa555/BTV-System-Design/actions/workflows/deploy.yml/badge.svg)](https://github.com/biggerbababa555/BTV-System-Design/actions/workflows/deploy.yml)

## 📌 Overview

BTV System Design is a dedicated documentation website and UI component library built for developer teams across Boonthavorn. It establishes unified standards for brand colors, typography, slim UI controls, iconography, toast alerts, and responsive layouts.

## 🚀 Built With

- **Framework:** Next.js 16 (App Router)
- **UI & Styling:** Tailwind CSS v4 & shadcn/ui
- **Icons:** Lucide Icons (`lucide-react`)
- **Toast Notifications:** Sonner (`sonner`)

## 🛠️ Local Development

```bash
# Clone the repository
git clone https://github.com/biggerbababa555/BTV-System-Design.git
cd BTV-System-Design

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) (or `http://localhost:3005`) to view the application.

## 📄 Documentation File

Developers can download the complete English design system specification directly from the website or access it in [`public/system-design.md`](public/system-design.md).
